黑苹果声卡驱动
2020-10-02

写在前面

安装完黑苹果后，便是各个硬件的驱动了。这篇就来介绍一下Clover引导的声卡驱动的两种方法，应该就能驱动市面上绝大部分的声卡了。下面就来看看吧。

材料用具

  1. Clover Configuration
  2. Apple ALC.kext
  3. Lilu.kext
  4. LayoutID速查表
  5. VoodooHDA.kext
  6. AIDA64

方法1：AppleALC驱动

  1. 确保VoodooHDA.kext已被卸载，一般在s\l\e目录下或者在这两个文件夹中：

    还有就是要注意s\l\e目录下的AppleHDA.kext是否健在，如果之前删了要恢复回去。

  2. 把下载好的AppleALC.kext和Lilu.kext丢进Others文件夹（或者对应OSX版本的文件夹也行）。

  3. 查找一下声卡型号，可以用AIDA64，也可以直接找，具体方法百度（加上关键词“ALC”可以更快找到）。找到声卡型号后，在速查表里面找一下有没有，一般都会有的。

  4. 用Clover Configuration打开config.plist，在这里注入相应的LayoutID。一个不行就试另一个，每次改完要重启查看是否驱动成功。

  5. 驱动成功就能看到这里有声音设备了。

方法2：Voodoo万能驱动

直接把下载好的VoodooHDA.kext扔进Others文件夹或对应版本的文件夹，重启一下，看看是否成功驱动（此方法作为备用方法，方法1不行才使用该方法）

优点&缺点

  1. Apple ALC驱动更加接近原生驱动，更加稳定，可能会有点麻烦（其实也不难），但用起来比较顺手。
  2. 有些型号没有LayoutID，想用Apple ALC只能自己用Linux调教，可以看看黑果小兵的文章。
  3. Voodoo虽简单，但是不稳定，需要在偏好设置里手动切换声音设备，有时可能会有爆音。
  4. 总之，推荐方法1，不能成功时才使用方法 2
